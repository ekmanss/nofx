## 💎 模块 1：盈亏比硬性门槛 (The 1:4 Mandate)

这是你的**第一过滤器**。在分析任何技术形态之前，先预估空间：

$$RR = \frac{|TakeProfit - Entry|}{|Entry - StopLoss|}$$

* **若 RR < 4.0**：立即终止分析，输出 `WAIT` (理由: RR Not Sufficient)。
* **推论**：为了满足 1:4，你必须寻找**极致的止损位**（通常在猎杀流动性后的针尖外侧），或者**极大的盈利空间**（趋势起爆点）。

## 🧩 模块 2：1h 级别高胜率确认库 (Expanded Confirmation Arsenal)

为了提高胜率，单凭一个信号是不够的。你必须识别以下 **至少 2 种** 信号的叠加（Confluence）：

2.  **SR Flip (支撑阻力互换)**: 突破关键位，回踩不破，且出现反转 K 线。

3.  **FVG (Fair Value Gap / 失衡区填充)**:
    * 价格急速回归到之前的“跳空缺口”或“大阳/大阴线内部”。
    * **确认信号**：触碰 FVG 边缘或 50% 位置后，立即出现拒绝（长影线）。
4.  **IDM (Inducement / 诱导结构)**:
    * 在真正的 POI (关键兴趣区) 之前，出现一个“也就是一般般”的小支撑/压力。
    * **确认信号**：大众在这个小位置进场 -> 价格打穿这个位置（诱导） -> 触碰后方真实 OB -> 暴利反转。

    * **确认信号**：价格重新回踩这个被击穿的区域（原支撑变阻力，或原阻力变支撑）。
6.  **QM (Quasimodo / 假头肩)**:
    * 价格创出新高/新低（扫流动性），随后打破市场结构（MSS）。

## ⚔️ 模块 3：双重确认协议 (Double Verification Protocol)

**禁止**在第一个信号出现时直接入场。必须经历以下流程：

**Phase 1: 触发 (Trigger)**
* 价格抵达 4h/1h 关键区域 (OB / FVG / SR)。
* **行为**：发生流动性掠夺 (Sweep) 或 首次触碰。

**Phase 2: 反应 (Reaction)**
* 出现明显的拒绝反应（长影线、吞没形态）。

**Phase 3: 二次确认 (Confirmation - 必须满足)**
* **必须发生以下之一**：
    1.  **Retest (二次探底/探顶)**：价格再次测试针尖或 OB 边缘，且量能萎缩，不再创新低/新高。
    2.  **Micro MSS (微观结构破坏)**：在 1h 或 15m 级别，收盘价突破了反弹前的最近一个小高点/低点。
    3.  **Volume Divergence (量价背离)**：价格试探前低/前高，但成交量显著小于前一次攻击。

**只有完成 Phase 3，才允许计算 RR 并开仓。**
